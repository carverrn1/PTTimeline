1Supported File Formats

PTTView can open the following image and document formats. The format is determined by the file extension.

Format Extensions Notes
SVG .svg, .svgz Vector format; rendered by the Qt SVG engine. Some advanced SVG features (CSS filters, complex gradients) may not render correctly due to Qt limitations. The DOCTYPE declaration is stripped automatically on load to prevent Qt from attempting a network fetch.
PNG .png Lossless raster format. Recommended for high-quality chart exports.
JPEG .jpg, .jpeg Lossy raster format. Suitable for screen viewing; not recommended where text sharpness is critical.
Bitmap .bmp Uncompressed raster format.
GIF .gif Raster format; only the first frame is displayed.
TIFF .tif, .tiff Raster format commonly used in print workflows.
WebP .webp Modern raster format; requires a Qt build with WebP support.
PDF .pdf Requires the PyMuPDF library (pip install pymupdf). Only the first page is rendered. Zoom quality is recalculated dynamically from the zoom percentage, capped at 600 DPI.
Note: If PyMuPDF is not installed, attempting to open a PDF file will display an error message. Install it with pip install pymupdf and restart PTTView.
SVG rendering: Qt's SVG renderer does not implement the full SVG 1.1 specification. Plots exported from PTTPlot are designed to be compatible, but third-party SVG files may not display correctly. For complex SVGs, export to PNG instead.

2Interface Overview

The PTTView window is composed of three main areas:

Image Canvas

The central area of the window. The loaded image is displayed on a scrollable canvas. When a file is first opened, the image is automatically scaled to fit the window (Fit to Window mode). The canvas supports both mouse-driven panning via the scroll wheel and keyboard-driven panning via the arrow keys.

Status Bar

A thin bar at the bottom of the window that provides two pieces of information at all times:

my_timeline_chart.svg     Press F1 for help Zoom: 85%
  • Bottom-left: The filename of the currently loaded image, followed by a reminder that F1 opens the help dialog.
  • Bottom-right: The current zoom level as a percentage (for example, Zoom: 85%).

Window Title Bar

When a file is loaded, the window title changes from PTTView to PTTView — filename, where filename is the base name of the loaded file. This makes it easy to identify the file when PTTView is running alongside other applications.

Menu Bar

Contains two menus:

  • File — Open a file, or exit the application.
  • Help — Access the User Guide, the Instructions dialog, the About box, and the System Information dialog.

Toolbar

A toolbar below the menu bar provides quick access to the most common zoom and view operations as clickable buttons. The toolbar is always visible and cannot be hidden.

ButtonEquivalent ShortcutAction
Zoom In (+) Ctrl++ Increase zoom by 15%.
Zoom Out (-) Ctrl+- Decrease zoom by 15%.
Fit to Window Ctrl+0 Scale the image to fit the current window size.
Actual Size Ctrl+1 Reset zoom to 100% (1:1 pixel mapping for raster images).
Scale… Ctrl+Shift+S Open the custom zoom dialog to type a specific zoom percentage.
Center Ctrl+Shift+C Centre the image within the visible canvas at the current zoom level.
Map Ctrl+M Open the Map Navigator thumbnail dialog.

3Launching PTTView

From the Command Line

PTTView accepts an optional image file path as a command-line argument:

python pttview.py [image_file]

Examples:

# Open a specific SVG file
python pttview.py my_timeline.svg

# Open a PNG export
python pttview.py output/project_chart.png

# Start with an empty viewer (no file loaded)
python pttview.py

Any supported file format can be passed on the command line. If the file path contains spaces, enclose it in quotes.

From PTTPlot

PTTView is designed to be launched automatically by PTTPlot after a chart is exported. When you use the Save & View option in PTTPlot, PTTPlot saves the plot file and then opens PTTView to display the result. No manual command-line invocation is required in this workflow.

Starting with an Empty Viewer

If no file argument is provided, PTTView opens with an empty canvas. Use Ctrl+O to open a file via the file dialog.

4File Operations

File operations are available from the File menu.

Operation Shortcut Description
Open Ctrl+O Opens a file dialog filtered to all supported image formats. Select any supported file to load it into the viewer. If a file is already open, it is replaced by the new selection.
Exit Ctrl+Q Closes the PTTView application. Equivalent to pressing Escape or closing the window via the title bar.

File Dialog Filter

The Open file dialog presents a combined filter for all supported formats, so you can navigate to any supported image file without adjusting the filter. Individual format filters are also provided for convenience.

Note: PTTView does not modify or write to any file. It is a read-only viewer. Opening a file simply loads it into the display canvas.

5Zoom Operations

PTTView supports a zoom range of 5% minimum to 500% maximum. The current zoom level is always shown in the status bar at the bottom-right of the window.

Operation Keyboard / Mouse Notes
Zoom in 15% Ctrl++ or Ctrl+= Increases zoom by 15 percentage points, centred on the middle of the visible canvas area.
Zoom out 15% Ctrl+- Decreases zoom by 15 percentage points, centred on the middle of the visible canvas area.
Fit to window Ctrl+0 Scales the image so that it fits within the current window size with a 20-pixel margin on all sides. This is the default zoom level when a file is first opened.
Actual size Ctrl+1 Resets zoom to 100% (1:1 pixel mapping for raster images). Useful for checking fine detail.
Custom zoom Ctrl+Shift+S Opens a dialog where you can type any zoom percentage from 5 to 500. The image is recentred after the zoom is applied.
Mouse wheel zoom Ctrl+Mouse Wheel Zooms in or out while keeping the point currently under the mouse cursor stationary in the viewport. The image pivots around the cursor position for precise, intuitive zooming.

PDF Zoom Quality

For PDF files, PTTView does not simply scale a fixed raster image. Instead, it re-renders the PDF page at a resolution derived from the current zoom level. The render DPI is calculated proportionally to the zoom percentage and is capped at 600 DPI to balance visual quality with memory and performance. Zooming in on a PDF therefore produces a progressively sharper image rather than an upscaled one.

Zoom Limits

Zoom is clamped at the boundaries: attempting to zoom below 5% or above 500% has no further effect. The status bar always reflects the current zoom percentage.

Tip: Use Ctrl+0 (Fit to Window) any time you want a full overview of the chart after zooming into a detail area.

6Panning & Scrolling

When the image is larger than the visible window (that is, when zoomed in beyond Fit to Window), you can pan to explore different regions using the mouse wheel or the keyboard arrow keys.

Gesture / Key Action
Wheel Up / Wheel Down Pan up or down through the image.
Shift+Mouse Wheel Pan left or right through the image.
↑ Arrow Up Pan the view upward.
↓ Arrow Down Pan the view downward.
← Arrow Left Pan the view to the left.
→ Arrow Right Pan the view to the right.
Note: Panning is only meaningful when the image extends beyond the visible window. At Fit to Window zoom level, the entire image is already visible and scrolling has no effect.

7Map Navigator

The Map Navigator provides a birds-eye thumbnail overview of the full image. It is especially useful when zoomed into a detail area and you want to jump to a different region without manually scrolling.

Opening the Navigator

Press Ctrl+M to open the Map Navigator dialog. There is no menu item for this feature; it is accessible by keyboard only.

Using the Navigator

  • The dialog displays a scaled-down thumbnail of the full image.
  • The thumbnail is scaled so that its longest side is at most 400 pixels, preserving the original aspect ratio.
  • Click anywhere on the thumbnail to instantly centre the main view on that location in the image.
  • The main canvas scrolls immediately to bring the clicked region into view.

Non-Blocking Dialog

The Map Navigator dialog is non-blocking — you can leave it open while continuing to work in the main window. Click different spots on the thumbnail repeatedly to navigate around the image. Close the dialog when you no longer need it.

Tip: The Map Navigator is most useful at high zoom levels (above 100%) where only a small portion of a large chart is visible in the main window. Click directly on the region of interest in the thumbnail to jump there immediately.

8View Controls

Additional view controls are available via keyboard shortcuts to manage the image position and display mode.

Operation Keyboard Description
Centre image Ctrl+Shift+C Centres the image within the visible canvas at the current zoom level. Useful after panning to bring the image back to a centred position without changing the zoom.
Toggle fullscreen F11 Switches the PTTView window between normal windowed mode and fullscreen. Press F11 again to return to windowed mode. The image is automatically reflowed to fit the new window size.

Fullscreen Mode

Fullscreen mode maximises the available canvas area by hiding the operating system title bar and taskbar. The menu bar and status bar remain visible. This is useful for presenting or reviewing a chart on a large display. All keyboard shortcuts continue to function while in fullscreen mode.

Tip: Press Ctrl+Shift+C after toggling fullscreen to re-centre the image in the newly expanded canvas area.

10Help Menu

The Help menu provides four items with reference and diagnostic information.

User Guide

Opens this User Guide HTML file in the system default web browser. If the file cannot be found at its expected path, a warning dialog is displayed.

Instructions  F1

Opens a concise in-application reference listing all controls and keyboard shortcuts. This is the quickest way to recall a shortcut without leaving PTTView. The same dialog is shown when pressing F1 at any time from the main window.

About

Displays the PTTView About dialog, which includes:

  • Application version number
  • Author and company name
  • Copyright notice

System Information

Displays a detailed system information dialog that is useful for diagnosing environment issues or reporting bugs. It includes:

  • Python version and interpreter path
  • Versions of key installed modules (PySide6, PyMuPDF, and others)
  • Operating system name and version
  • Relevant file paths (application directory, library directory)
Tip: Copy the contents of the System Information dialog when reporting bugs, as it captures the exact environment in which PTTView is running.
